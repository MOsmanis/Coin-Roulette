Terminal game made with Python library Curses.
Made in 3 hours for Trijam18 (https://itch.io/jam/trijam-18)
https://skurstenis.itch.io/coin-roulette

1. Install Curses for Python (run "python package_install.py")
2. Play the game by running "python CoinRoulette.py" in your console

Window needs to be big enough, otherwise it will crash
